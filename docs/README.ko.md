이 저장소가 해결한다. AI가 당신이 누구인지, 어떤 결정을 내렸는지, 무엇을 배웠는지, 무엇을 싫어하는지 기억한다. 모든 도구에서. 모든 세션에서. 영원히.

## 설정 (몇 초)
