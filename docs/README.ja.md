## セットアップ（10秒）

**ステップ1**: **"Use this template"** → **Create a new repository** → **Private** に設定

**ステップ2**: ステップ2はない。

任意のAIツールでこのリポジトリを開く。指示ファイルを読み、いくつか質問し、プロファイルの構築を開始する。以降、AIはあなたを覚えている。
